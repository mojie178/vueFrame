# 慧联运平台

## src 文件说明
```
+ api           接口文件
+ components    组件文件
+ config        配置文件
+ directives    公共指令
+ filters       公共过滤器
+ images        图片文件
+ mixins        混入文件
+ mock          模拟数据
+ router        公共路由文件
+ store         vuex文件
+ styles        公共样式文件
+ util          公共方法文件
+ views         页面视图文件
```

## Project setup
```
yarn install
```
